Abstract—Transformer-based Large Language Models (LLMs)
possess advanced language understanding capabilities, but their
performance often degrades when processing highly extended
contexts. Recent advancements, such as the ReadAgent system,
have attempted to mitigate this by implementing an episodic
gist memory and an interactive lookup mechanism. However,
this system retains fundamental limitations: it does not grant
significantly higher context lengths inherently, the sequential
lookup strategy incurs high computational latency, and the
reliance on unconditional gisting increases the risk of halluci-
nation when verbatim text is excluded. This paper proposes a
theoretical framework to resolve these constraints through four
novel mechanisms: a recursive ”FractalAgent” for higher context
scaling, a predictive task-driven gisting protocol to preserve
critical details, a multimodal extension termed ”WatchAgent”
for long-form video, and a differentiable gist retrieval system
designed to bypass inference overhead. By synthesizing these
approaches, this work aims to establish a more scalable, efficient,
and versatile architecture for long-context understanding.
I. INTRODUCTION
Transformer-based Large Language Models (LLMs) are
highly capable of language understanding, but the amount of
text that they are able to read at one time is constrained.
Not only is there an explicit context length limitation, but it
has also been found that the performance of LLMs tends to
decline with increasingly long inputs, even when they do not
actually exceed the explicit context window. To address these
inherent differences in reading approaches between machines
and humans, systems like ReadAgent have been proposed
to compress memory episodes into short episodic memories
called gist memories. While this approach extends effective
context length, it does not give substantially higher context
lengths without limits, nor does it guarantee good performance
when the gist memory itself is extremely long.
A primary challenge in this domain is that the system breaks
down if the concatenated gist memory itself exceeds the un-
derlying model’s context window, as existing methods cannot
process inputs beyond this hard limit. Additionally, because
many details are elided in the gist memories, if the model is
called upon to perform a task requiring those details, it may
generate them itself without giving any indication, leading to
an increased tendency to hallucinate. Furthermore, the larger
number of interactions with the model required for sequential
look-up significantly increases the computational cost. Lastly,
unconditional gisting, where the text is compressed without
knowing the final task, may produce broadly useful gists at
the cost of reduced compression and increased distracting
information.
To overcome these barriers, the focus of this paper is to
propose four novel architectural directions. First, to address
the breakdown when gist memory exceeds context limits, we
propose a recursive gisting mechanism (”FractalAgent”). By
creating a ”gist of the gists,” the model can navigate a fractal
tree during the interactive lookup phase, theoretically allowing
for longer context processing. Second, to mitigate the loss of
critical information and reduce hallucinations, we introduce
a predictive task-driven gisting system. A forecasting agent
generates hypothesized tasks based on text type, condition-
ally compressing irrelevant sections while retaining verbatim
detail in flagged areas. Third, we propose extending this
episodic gisting framework beyond text into multimodal long-
form video understanding (”WatchAgent”), utilizing vision-
language models to paginate and gist video scenes for targeted
high-frame-rate retrieval. Finally, to eliminate the computa-
tional latency of sequential prompting, we propose a differen-
tiable gist retrieval system. By training an adapter layer, the
agent can bypass prompt overhead and utilize cross-attention
over gist representations to automatically retrieve raw pages
during the forward pass.

II. LITERATURE REVIEW
The challenge of processing long contexts in Large Lan-
guage Models has been approached from multiple architec-
tural and methodological directions. While recent hardware
advancements have explicitly increased context windows, re-
search indicates that LLM performance tends to decline with
increasingly long inputs, particularly struggling to retrieve
information located in the middle of extended documents
[1]. To mitigate the computational overhead of processing all
tokens simultaneously, efficient fine-tuning methods such as
LongLoRA have been proposed to extend context sizes with
minimal computational cost [2]. Furthermore, architectural in-
novations like RingAttention compute self-attention blockwise
across multiple devices, theoretically enabling near-infinite
context processing [3]. However, scaling the context window
alone is insufficient if the model cannot effectively isolate
and reason over nuanced information without performance
degradation.
To bypass the need for massive contiguous context win-
dows, research has increasingly shifted toward structured,
agentic memory frameworks and interactive reasoning. The
ReAct framework established foundational methods for in-
terleaving reasoning and environment actions, allowing mod-
els to iteratively gather relevant information [4]. Building
upon iterative reasoning, systems like MemWalker read long
documents interactively by traversing a tree of hierarchical
summaries, enabling the model to navigate beyond its explicit
context limit [5]. For longer-term contextual preservation,
frameworks such as MemGPT manage an LLM’s memory
akin to an operating system, dynamically paging information
between short-term and long-term storage to handle effectively
infinite contexts [6]. Similarly, the Generative Agents architec-
ture demonstrated that maintaining a comprehensive memory
stream of experiences, coupled with periodic synthesis and
retrieval, significantly enhances an agent’s ability to maintain
logical consistency over extended periods [7].
More recently, addressing human-inspired reading method-
ologies specifically, the ReadAgent system demonstrated that
long-document comprehension could be significantly im-
proved by parsing texts into manageable episodes and sub-
sequently compressing them into gist memories [8]. While
these studies collectively establish foundational methods for
long-context retrieval, gist memory, and iterative reasoning, a
significant gap remains regarding higher context scaling, mul-
timodal adaptation, and the computational latency of sequen-
tial prompting. The theoretical frameworks proposed in this
paper aim to directly address these unfulfilled requirements
by building upon these recent agentic memory concepts.
